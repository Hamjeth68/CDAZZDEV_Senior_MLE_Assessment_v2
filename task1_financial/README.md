# Task 1: Financial AI Equity Research Assistant

## Objective
Implement Task 1 notebook and supporting source modules for OHLCV ingestion, indicator computation, news sentiment, and recommendation generation.

## Structure
- `src/` - pipeline and service modules
- `outputs/` - generated markdown/html/chart artifacts
- `task1_equity_research.ipynb` - executed notebook (outputs must remain visible)

## Run
1. Install root requirements.
2. Open `task1_equity_research.ipynb`.
3. Execute all cells and verify output artifacts under `outputs/`.
